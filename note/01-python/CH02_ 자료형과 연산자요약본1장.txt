1절 변수
- 파이썬은 변수선언을 위한 과정이 없음. 할당과 동시에 타입이 지정되고 값이 할당됨
1.1 변수할당
- 할당연산자(=)를 이용해서 값을 할당하면 다입이 셋팅
- 파이썬에서의 모든 변수는 객체로 저장
1.2 변수이름 규칙
- 문자로 시작 (숫자시작불가)
- 문자, 숫자, _로 변수명 작성
- 함수명은 변수이름으로 가능하나 피한다. 써야한다면 int_식으로 _를 이용
- 파이썬 예약어(for, ..)는 변수이름 사용 불가
- 한글변수는 꼭 필요시에만 이용
1.3 도움말
- help(), shift+tab키 tap키 2번누르면 길게
1.4 다중 변수 할당
- 다중 변수 할당시 순차적 할당이 아니고, 일괄 동시 할당
2절. 화면입출력(i/o)
- input()은 입력된 값을 문자로 반환
- print(출력 내용, sep=' ', end='\n')
3절. 기본 자료형
- 자료형에 따라 값의 크기와 저장방식이 달라짐
	- 숫자 : int, float, complex
	- 문자 : str
	- 논리 : bool(True/False)
3.1 숫자형
- 숫자형 : int, float, complex
	 int값/int값=>실수
	 지수형 표현(큰수, 작은수)은 => 실수
3.2 문자형
- 문자형 : str
	문자열 연산 : +(연결) *(반복)
※ 문자열 연산
- 여러 줄 주석 용도
※ raw 데이터
- 용도 : 정규표현식 이용, 디렉토리 경로 표현시 이용
